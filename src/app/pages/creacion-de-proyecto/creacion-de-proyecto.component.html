Proceso de Creación de un Proyecto en Angular

Para comenzar un nuevo proyecto en Angular, sigue estos pasos:

  1. Instalar Angular CLI:

    Para instalar Angular CLI, asegúrate de tener Node.js instalado y ejecuta:

    npm install -g angular/cli
  2. Crear un nuevo proyecto:

    Usa el siguiente comando para crear un nuevo proyecto:

    ng new nombre-del-proyecto

    Elige las opciones de configuración durante el proceso (enrutamiento, estilos, etc.).

  3. Navegar al directorio del proyecto:

    Entra en la carpeta del proyecto con el comando:

    cd nombre-del-proyecto
  4. Ejecutar la aplicación:

    Inicia el servidor de desarrollo ejecutando:

    ng serve

    Luego, abre http://localhost:4200 en tu navegador para ver la aplicación en ejecución.

  5. Personalizar la aplicación:

    Edita el archivo src/app/app.component.html para cambiar el contenido de la página principal.
