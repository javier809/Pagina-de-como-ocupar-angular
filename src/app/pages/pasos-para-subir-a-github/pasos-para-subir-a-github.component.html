Pasos para Copiar el Código en Git

Sigue estos pasos para versionar tu código en Git:

  1. Inicializa un Repositorio Local:

    Para empezar, dirígete a la carpeta de tu proyecto y ejecuta:

    git init
  2. Añade todos los archivos al Repositorio:

    Para agregar todos los archivos al repositorio, ejecuta:

    git add .
  3. Realiza el Primer Commit:

    Guarda los cambios con un mensaje descriptivo usando:

    git commit -m "Primer commit"
  4. Conecta a un Repositorio Remoto:

    Vincula tu repositorio local a un repositorio remoto utilizando:

    El mismo Gitgub te Proporciona un URL

    git remote add origin URL_DEL_REPOSITORIO
  5. Sube Cambios al Servidor:

    Finalmente, sube tus cambios al repositorio remoto:

    git push -u origin main
