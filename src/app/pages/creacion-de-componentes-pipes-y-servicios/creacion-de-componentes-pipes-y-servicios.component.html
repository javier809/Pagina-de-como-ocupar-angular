Creación de Páginas, Componentes, Pipes y Servicios en Angular

Angular permite crear varios elementos fácilmente mediante la CLI. A continuación, se muestran los comandos para generar cada uno:

  • Componentes:

    Para crear un nuevo componente, utiliza el siguiente comando:

    ng generate component nombre-componente
  • Páginas (con enrutamiento):

    Crear un componente y añadir la ruta correspondiente en el módulo de enrutamiento para que se pueda navegar a la nueva página:

    ng generate component nombre-pagina

    A continuación, agrega la ruta en el archivo de enrutamiento:

    path: 'nombre-pagina', component: NombrePaginaComponent
  • Pipes:

    Para crear un pipe, ejecuta el siguiente comando:

    ng generate pipe nombre-pipe

    Luego, registra el pipe en el módulo correspondiente.

  • Servicios:

    Para generar un nuevo servicio, usa este comando:

    ng generate service nombre-servicio

    El servicio estará listo para ser inyectado en los componentes o módulos necesarios.
